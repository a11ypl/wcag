CSS

Style nie mogą zabierać użytkownikowi informacji, fokusu ani możliwości powiększenia treści.

Instrukcja praktyczna

Co trzeba sprawdzić?

  • Czy kontrast tekstu i elementów interaktywnych jest wystarczający.
  • Czy fokus jest widoczny i nie jest usuwany bez zamiennika.
  • Czy treść działa przy powiększeniu i zmianie odstępów tekstu.
  • Czy układ nie wymaga przewijania w dwóch kierunkach przy typowym powiększeniu.
  • Czy hover nie jest jedynym sposobem ujawnienia treści.
  • Czy animacje respektują prefers-reduced-motion.

Polecane zasoby

  1. Sprawdź wyjaśnienie WCAG 2.2 dla kryterium Reflow (otwiera się w nowym oknie)
  2. Przejdź do wyjaśnienia WCAG 2.2 o kontraście elementów nietekstowych (otwiera się w nowym oknie)
  3. Otwórz wyjaśnienie WCAG 2.2 dotyczące wyglądu fokusu (otwiera się w nowym oknie)
